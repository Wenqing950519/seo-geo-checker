# 最終 CTA 與 Lead 表單

## 區塊目標

在使用者已經理解痛點、方法、範例報告與服務後，引導他採取下一步。

## CTA 版本 A：免費健檢

標題：

```text
想知道 AI 眼中的你的網站是什麼樣子？
```

副標：

```text
輸入網址，取得一份初步 SEO/GEO 健檢報告，了解你的網站是否被搜尋引擎與 AI 正確理解。
```

按鈕：

```text
開始免費健檢
```

## CTA 版本 B：預約診斷

標題：

```text
看完報告後，不確定該怎麼修？
```

副標：

```text
留下你的網站與聯絡方式，我們可以協助你解讀健檢結果，判斷最值得優先處理的 SEO/GEO 問題。
```

按鈕：

```text
預約一次 SEO/GEO 診斷
```

## Lead 表單欄位

必填：

- Email
- Website URL

選填：

- 想解決的問題

## 表單欄位文案

Email：

```text
you@company.com
```

Website URL：

```text
https://your-website.com
```

需求描述：

```text
例如：想知道為什麼 AI 搜尋都推薦競品，沒有推薦我們
```

送出按鈕：

```text
送出，預約診斷
```

## 表單下方說明

```text
我們會根據你提供的網站與需求，回覆初步建議。若需要完整 SEO/GEO 優化，會再提供客製化合作方式。
```

## 表單成功訊息

```text
已收到你的需求。我們會盡快查看網站資訊，並回覆下一步建議。
```

## 表單失敗訊息

```text
送出時發生問題，請稍後再試，或直接改用 Email 聯繫。
```
