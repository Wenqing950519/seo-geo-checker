# 聲明與 FAQ

## 區塊目標

降低使用者疑慮，避免過度承諾，同時讓產品看起來更可信。

## 透明聲明

```text
這是一份初步 SEO/GEO 健檢，不等於完整 SEO audit 或保證排名方案。報告會根據網站公開資訊、技術訊號與 AI/search 測試結果整理初步建議。AI 搜尋結果可能受到模型、時間、地區、搜尋環境與資料來源影響，因此結果應視為診斷訊號，而非固定排名或保證結果。
```

## FAQ 1：這份健檢是免費的嗎？

回答：

```text
是。第一版健檢可以免費使用，用來快速了解網站在 SEO/GEO 上可能存在的問題。若你需要完整優化方案或人工解讀，可以再預約進一步診斷。
```

## FAQ 2：我需要註冊帳號嗎？

回答：

```text
不需要。輸入網址即可產生初步健檢報告。若你希望我們協助解讀或提供後續優化建議，再留下 Email 與需求即可。
```

## FAQ 3：這份報告會保證我的網站被 AI 引用嗎？

回答：

```text
不會。AI 搜尋結果會隨模型、時間、地區與搜尋環境變動，沒有人能保證特定網站一定被 ChatGPT、Perplexity 或 Gemini 引用。這份健檢的目標是找出影響 AI 理解與引用機會的問題，並提供優先修正方向。
```

## FAQ 4：GEO 和 SEO 有什麼不同？

回答：

```text
SEO 主要關注搜尋引擎如何爬取、理解、索引與排名網站。GEO 則更關注網站內容是否容易被生成式 AI 理解、引用與整合進答案。兩者不是互相取代，GEO 通常建立在穩固的 SEO、內容品質與品牌實體訊號之上。
```

## FAQ 5：健檢會讀取我的網站後台資料嗎？

回答：

```text
不會。初步健檢只使用公開可讀取的網站資訊，例如首頁內容、metadata、robots.txt、sitemap、schema 與公開頁面內容。若需要結合 Search Console 或網站後台數據，會在後續顧問服務中另外處理。
```

## FAQ 6：為什麼報告結果可能和我實際看到的不一樣？

回答：

```text
AI 搜尋與一般搜尋結果都可能受到時間、地區、登入狀態、模型版本與搜尋方式影響。因此報告會把結果視為診斷訊號，而不是絕對排名。真正的優化仍需要搭配多次測試、競品觀察與網站數據。
```

## FAQ 7：免費報告和完整顧問服務差在哪？

回答：

```text
免費報告提供初步問題、優先級與修正方向。完整顧問服務會進一步結合商業目標、Search Console 數據、競品研究、內容策略與技術實作建議，產出更完整的優化方案。
```

## FAQ 8：我的網站資料會被公開嗎？

回答：

```text
報告頁預設不應被搜尋引擎索引，也不會主動公開你的網站分析結果。若未來要展示案例，會先匿名化或取得同意。
```
