# 服務方案區

## 區塊目標

讓免費健檢自然導向顧問服務。這一區不能只列服務名稱，要用買方情境描述。

## 區塊標題

```text
發現問題之後，我們幫你把它修成可執行的優化方案
```

## 區塊副標

```text
免費報告讓你看見問題；顧問服務則協助你判斷優先級、補齊內容與技術訊號，讓網站更容易被搜尋引擎理解、被 AI 正確引用。
```

## 服務 1：GEO 優化服務

標題：

```text
GEO 優化服務
```

適合誰：

- AI 搜尋常推薦競品，但沒有推薦你
- 網站有內容，但 AI 無法清楚理解你的定位
- 想提升品牌在 ChatGPT、Perplexity、Gemini 類搜尋中的出現機會
- 想知道該如何布局 FAQ、比較頁、案例頁與可引用內容

包含：

- AI 眼中定位診斷
- 競品敘事分析
- 內容可引用性優化
- FAQ / comparison / evidence page 規劃
- schema 與品牌實體訊號建議

CTA：

```text
預約 GEO 診斷
```

## 服務 2：SEO 顧問服務

標題：

```text
SEO 顧問服務
```

適合誰：

- 想建立長期自然搜尋流量
- 不知道該先做內容、技術還是網站架構
- 需要 Search Console 數據解讀
- 想把 SEO 從零散文章升級成完整策略

包含：

- SEO 策略規劃
- 關鍵字與搜尋意圖分析
- 網站架構建議
- 內容主題地圖
- Search Console 數據解讀
- 優先級與執行路線圖

CTA：

```text
預約 SEO 諮詢
```

## 服務 3：技術 SEO 健檢

標題：

```text
技術 SEO 健檢
```

適合誰：

- 網站改版前後
- 收錄或流量異常
- 想檢查 indexability、schema、canonical、sitemap、內部連結
- 網站架構複雜，需要釐清搜尋引擎是否能正確讀取

包含：

- Indexability 檢查
- robots.txt / sitemap / canonical 檢查
- schema / structured data 建議
- 內部連結與頁面層級檢查
- Core Web Vitals 初步方向
- 技術問題優先級排序

CTA：

```text
預約技術健檢
```

## 服務區補充說明

```text
目前服務以客製化診斷與優化建議為主。每個網站的 SEO/GEO 問題都不同，因此會先根據健檢結果與商業目標，判斷最值得優先處理的方向。
```
